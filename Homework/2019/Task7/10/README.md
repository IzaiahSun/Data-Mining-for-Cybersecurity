# webshell detection
- #### webshell:

  - 基于互联网web程序以及web服务器而存在的一种后门形式，主要通过网页脚本程序和服务器容器所支持的后端程序，在web服务器及其中间件中进行运行。

- #### 实验环境

  - matplotlib

  - seaborn

  - sklearn

    

## 数据集
- 白样本（white_list）：共计3586条

- 黑样本（black_list）：共计2681条

- 训练集大小：5000  测试集大小：1249

  

## 特征选取
- 文件大小
- 信息熵
- 特殊函数出现次数
